ISSL Objective

To provide security solutions for security testing of web applications, enhancing the security of IoT devices and using blockchain technology for developing secure applications

ISSL Domains (Sub-Labs)

The Innovative Secured Systems Lab (ISSL) is further divided into the following sub-labs:


Secured IoT Devices

The primary objective of this part of the lab is to develop secure communication systems using IoT devices for remote monitoring via sensors network. The utilization of IoT devices for monitoring of goods during conveyance which has a paramount importance for ensuring the secure delivery of articles in good shape.


Blockchain Security

This lab focuses on using the Blockchain technology for developing secure web and mobile applications in various domains including gem & jewelers, health, groceries, financial sectors, using formal modelling and verification framework for blockchain technologies.


Security Testing

This lab aims to investigate the security of web applications/services against cyber-attacks that exploit Injection vulnerabilities. Developed effective security testing techniques/tools for the detection of vulnerabilities based on novel techniques (e.g., Evolutionary algorithms/Search-based testing/Genetic algorithm). The lab is also developing cost effective simulation environments for different social engineering attacks and application vulnerabilities.
