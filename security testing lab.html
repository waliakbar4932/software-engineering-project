Security Testing Lab

Security Testing lab aims to investigate the security of web applications/services against cyber-attacks that exploit Injection vulnerabilities. Developed effective security testing techniques/tools for the detection of vulnerabilities based on novel techniques (e.g., Evolutionary algorithms/Search-based testing/Genetic algorithm). The lab is also developing cost effective simulation environments for different social engineering attacks and application vulnerabilities.

Current Domains of Security Testing Lab



Developing Effective Vulnerability Tools

Our lab focuses on creating sophisticated tools to detect and neutralize vulnerabilities in web-based systems and mobile applications, ensuring robust digital protection.
